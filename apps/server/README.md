# Node.jsによるノートアプリAPI実装

`apps/web`で利用するAPIサーバー。

## 開発

```bash
# install dependencies
vp install

# run lint, fmt and type check
vp check --fix

# run tests
vp test

# run the development server
vp run dev      # not `vp dev` because it should run the npm script

# build the app
vp run build   # not `vp build` because it should run the npm script
```
